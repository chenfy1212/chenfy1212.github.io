# 木沐景观宣传网站

这是一个不依赖境外字体、地图或 CDN 的纯静态网站，可部署到中国大陆可访问的静态托管服务。

## 本地查看

直接双击 `index.html` 即可查看。也可以在本目录启动任意静态文件服务器。

## 编辑内容

- 服务、案例分类、案例和服务流程都集中在 `content.js`。
- 新案例图片放入 `assets/images/`，再复制 `content.js` 中 `projects` 里的一条记录并修改。
- 页面主体文字在 `index.html`。
- 配色与版式在 `styles.css`。

## 上线前建议补充

- 已备案的大陆域名与服务器/对象存储。
- 页脚的完整公司主体信息与 ICP 备案号。
- 确认微信二维码仍为当前工作室对外联系二维码。
- 补充项目所在区域、面积、完成年份和项目故事，可提升案例说服力与搜索展示效果。
